# 🚗 Car Sales Dashboard (Power BI)

## 📊 Overview
This project analyzes car sales data to track performance across different regions, models, and time periods.

## 🛠 Tools Used
- Power BI
- Excel

## 📈 Key Features
- Total Sales, Revenue, and Profit KPIs
- Sales by Region
- Top Performing Car Models
- Monthly Sales Trends

## 🔍 Key Insights
- Highest sales observed in [City/Region]
- [Car Model] is the top-selling model
- Sales peak during [Month/Season]
